TITLE: IT Support Guidelines

CATEGORY: IT
PRIORITY: MEDIUM

CONTENT:
This document outlines IT support procedures applicable to both customers and employees.

For Common Users (Customers):
Customers may face issues while accessing the company’s website, mobile app, or online services.

Login Issues:
If a customer is unable to log in, they should first verify their username and password. If forgotten, they can use the "Forgot Password" option to reset their credentials via registered email or mobile number.

Website or Application Errors:
If a customer encounters errors such as pages not loading, payment failures, or app crashes, they should try refreshing the page, clearing browser cache, or reinstalling the app. Persistent issues should be reported via the support chat or ticket form.

Account Lock:
Accounts may get temporarily locked after multiple failed login attempts. Customers should wait for 15 minutes or contact support for manual unlocking.

Email Notifications Not Received:
Customers should check spam or junk folders. If emails are still not received, they should ensure the registered email address is correct.

For Employees:
Password Reset:
Employees can reset their system password using the "Forgot Password" option on the login page. A reset link will be sent to the registered email address.

System Slowness:
Employees should restart the device and close unused applications before raising a ticket.

Software Installation:
Employees must request software installation with manager approval. Unauthorized installations are prohibited.

Email Issues:
Employees facing official email access issues should verify credentials and network connectivity.

Hardware Issues:
Hardware issues should be reported with asset ID and issue description.
