TITLE: VPN Access Issues

CATEGORY: IT
PRIORITY: HIGH

CONTENT:
This document explains how users can resolve VPN access problems.

For Common Users (Customers):

- Ensure VPN client is installed and updated
- Verify username and password
- Check internet connectivity
- Restart VPN client
- Try connecting from a different network

Common VPN Errors:

- Authentication failed → reset password
- Connection timeout → check firewall/network
- Server unreachable → VPN maintenance issue

When to Contact Support:

- VPN fails after password reset
- Repeated authentication failures
- VPN client crashes or freezes

For Employees:

- VPN access requires manager approval
- New joiners receive VPN access within 24 hours
- Role changes require VPN access revalidation
